The Texter API Documentation

User Type

{
    Password: number,
    Username: string,
    ID: number,
    Following: number[],
    Followers: number[],
    Articles: string[]
}
  • Password is generated by hashing the real password with SHA256
  • ID is generated by getting the next free number
  • Following is an array of ID of users, followed by the user
  • Followers is an array of ID of users, following the user
  • Articles is an array of ID of articles, owned by the user
