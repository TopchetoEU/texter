The Texter API Documentation

/users/create - create new user out of thin air

  • Body:
    • Credentials: Credentials - user's credentials (See credentials type)
    • New: object - New username and password
      • Username: string - New title of the article
      • Password: string - New content of the article (will be hashed with sha256)
  • Result
    • Error: ErrorFormat - read about error format at this link
    • NewId: string - the generated ID for the user (generation method - while (userIds.indexOf(id) !== -1) { id = Math.random() * Number.MAX_SAFE_INTEGER; })
  • Possible errors
    • Missing property New - Make sure you're providing property New and you didn't misspelled it
    • Usual credentials errors - read about possible credentials errors
