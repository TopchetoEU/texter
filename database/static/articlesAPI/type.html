The Texter API Documentation

Type

{
    ID: number,
    CreatorID: number,
    Title: string[]
    Content: string,
    Likers: number[],
    Dislikers: number[],
}
  • ID is generated by hashing the Title with SHA256
  • CreatorID is the ID of the creator
  • Content is the content of the article
  • Likers is an array of IDs of users, liked the article
  • Dislikers is an array of IDs of users, disliked the article
