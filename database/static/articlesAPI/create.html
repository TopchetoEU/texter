The Texter API Documentation

/articles/create - create new article out of thin air

  • Body:
    • Credentials: Credentials - user's credentials (See credentials type)
    • New: object - New title and content
      • Title: string - New title of the article
      • Content: string - New content of the article
  • Result
    • Error: ErrorFormat - read about error format at this link
    • NewID: string - the generated ID for the article (generation method - sha256(title + content))
  • Possible errors
    • Missing property New - Make sure you're providing property New and you didn't misspelled it
    • Usual credentials errors - read about possible credentials errors
