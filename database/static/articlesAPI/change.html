The Texter API Documentation

/articles/change - change article, like or dislike it

  • Body:
    • Selector: object - the way you wanna find the item (see mongodb refrence)
    • Credentials: Credentials - user's credentials (See credentials type)
    • Like: Number - -1 stands for dislike and 1 stands for like
    • DoModify: boolean - Will a change will be executed
    • Modify: object - A list of modification settings
      • [Property name]: any - Property name - name of property for change, Value - value to set to the property
  • Result
    • Error: ErrorFormat - read about error format at this link
  • Possible errors
    • Missing property Selector - Make sure you're providing selector and you didn't misspelled it
    • Usual credentials errors - read about possible credentials errors
    • I'm a teapot - The server refuses the attempt to brew coffee with a teapot.
