The Texter API Documentation

/articles/get - get by selector

  • Body:
    • Selector: object - the way you wanna find the item (see mongodb refrence)
    • DoPaging: boolean - is the result is gonna be paged
    • Paging - all paging settings
      • PageSize: number - the size of the page
      • PageCount: number - the number of the page
  • Result
    • Found: Article[] - (read about article type)
    • Error: ErrorFormat - read about error format at this link
  • Possible errors
    • Missing property Selector - Make sure you're providing selector and you didn't misspelled it
